## `local에서 Node 셋팅하기`

### `Local환경에서 Node 서버 다른 사람도 접속하기`

AWS EC2를 아직 구매하지 못해서, 내 Local PC에서 개발을 하고

프론트엔드와 연동을 하기 위해 시간을 투자했었는데, 이후에도

까먹지 않도록 여기에 기록을 해둔다.

<br />

### `Node 코드`

먼저 모든 IP에서 접근을 가능하게 하려면 Node.js 코드를 수정해야한다.

<br />

```javascript

app.listen(3000, "0.0.0.0", () => {
    console.log('Server running on port 3000');
});


```

"0.0.0.0"은 모든 네트워크 인터페이스에서 요청을 수신한다.

<br />

### `url은?`

내 PC의 사설 IP를 확인

- Windows: ipconfig -> IPv4 주소 (예: 192.168.0.101)
- Mac/Linux: ifconfig 또는 ip addr

<br />

#### `내 컴퓨터 IP (사설 IP)`

- 같은 네트워크 안에서 다른 장치와 통신
- 외부 인터넷에서는 접근 불가
- **Postman에서 다른 PC가 내 서버에 접근하려면 이 IP 사용**

<br />

#### `회사 IP`

- 회사에서 인터넷에 접속할 때 외부에 노출되는 IP
- 인터넷 서비스 제공자(ISP)가 할당
- 회사 네트워크 전체를 인터넷에서 식별
- 회사가 외부에 노출되는 IP가 203.0.113.25라고 한다면, 인터넷에 접속할 때는 이 IP 사용

<br />

### `요약`

- 사설 IP = 내 PC 고유 번호, 내부 통신용
- 회사 IP = 회사 네트워크 번호, 외부에서 회사 전체를 대표
